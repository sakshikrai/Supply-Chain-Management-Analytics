# 🚢 Supply Chain & Inventory Management Analytics 

## 📋 Project Overview
This is an end-to-end data analytics project designed to evaluate supply chain efficiency, track profitability, and forecast demand. The project involves building a relational database from raw logistics data and developing interactive dashboards across multiple Business Intelligence platforms to extract actionable insights.

## 🛠️ Tech Stack & Tools Used
* **Database Management:** SQL Server (T-SQL)
* **Data Visualization & BI:** Power BI, Tableau Public, Microsoft Excel
* **Data Processing:** Pivot Tables, DAX, SQL Aggregate Functions, ETL Processes



## 📊 Key Insights & KPIs Analyzed
1. **Financial Metrics:** Total Sales Revenue, Average Order Value (AOV), and Total Cost of Inventory.
2. **Delivery Performance:** On-time vs. Delayed shipment ratio across major carriers (FedEx, DHL, BlueDart).
3. **Inventory Management:** Stock status, Inventory Value by Product Category, and Region-wise sales distribution.
4. **Demand Forecasting:** Trend analysis comparing Actual Demand vs. Forecasted Demand over time.
5. **Customer Demographics:** Order volume segmented by Customer Age Group, Region, and Purchase Method.

## 🚀 How to Use This Project Locally
1. Clone this repository to your local machine.
2. To view the BI files locally, open the `.pbix`, `.twbx`, or `.xlsx` files in their respective applications.
3. To recreate the database, execute the SQL scripts (inside the `SQL` folder) in order using SQL Server Management Studio (SSMS).
